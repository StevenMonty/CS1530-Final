## Run Locally
1. Run `./initEnv.sh`
1. Run `./runDjango.sh`
1. In a separate window, Run `./runReact.sh`

## Run with Docker
`docker-compose up|down|build`

## After running
View the API at localhost:8000<br>
View the frontend at localhost:3000
